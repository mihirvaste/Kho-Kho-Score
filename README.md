# Lightweight Login App

A dependency-free Python login page backed by SQLite.

## Run locally

1. Open PowerShell.
2. Run:

   ```powershell
   cd D:\score
   python app.py
   ```

3. Open http://127.0.0.1:8000 in a browser.
4. Sign in with username `admin` and password `admin`.
5. Press `Ctrl+C` in PowerShell to stop the server.

The `login.db` SQLite database is created automatically the first time the app runs.
